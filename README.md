# 🗺 프로젝트 소개
- 배포링크
  - 준비중😍
- 프로젝트 시작 계기
  - 내가 살고있는 동네를 소개해 보고 싶다!
  - 나만 알고 있는 동네의 핫플을 알리자!
- 프로젝트 내용
  - 지도에서 동네별 평점을 한 눈에 확인할 수 있는 서비스!
  - 확대/축소에 따라 동네 단위(도, 시/군/구, 동) 실시간 변경
  - 내가 살고 있는 동네의 평점과 후기를 작성하고 공유
  - 사진과 함께 내 동네의 맛집, 여행지, 편의시설 등을 소개하는 글을 작성하고 공유
